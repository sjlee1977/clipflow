# 쇼핑 쇼츠 스크립트 구조

## 15초 구조 (틱톡/릴스 기본형)

```
[0~3초]  훅       — 시청자를 멈추게 하는 한 문장 or 장면
[3~10초] 핵심시연  — 제품의 가장 놀라운 기능/결과 하나
[10~15초] CTA     — 구매/링크 유도 (짧고 직접적)
```

**예시 (세럼 제품):**
- 훅: "피부과 원장이 직접 쓰는 세럼이 있어요" (자막 + 제품 클로즈업)
- 핵심: 아침/저녁 비교샷 → 2주 전후
- CTA: "링크는 바이오에 있어요 ↑"

---

## 30초 구조 (표준형 — 가장 많이 쓰임)

```
[0~5초]   훅          — 공감 or 충격 or 궁금증
[5~10초]  문제 제시    — 시청자가 공감하는 고통
[10~22초] 제품 시연    — 사용 장면 + 결과 (시각적으로)
[22~27초] 사회적 증거  — "XX명이 재구매" or 별점
[27~30초] CTA         — 링크/할인코드
```

**예시 (청소 도구):**
- 훅: "이거 보고 나서 청소 방식이 바뀌었어요"
- 문제: 구석 먼지 못 치우는 모습
- 시연: 도구 사용 → 깨끗해지는 화면
- 증거: "후기 4.9점, 3만 개 판매"
- CTA: "지금 20% 할인 중 → 링크"

## 60초 구조 (스토리형 — 고관여 제품)

```
[0~5초]   훅          — 강렬한 오프닝 장면 or 질문
[5~13초]  문제 공감    — 구체적 상황 묘사 (시청자의 일상)
[13~33초] 제품 시연    — 사용 전/중/후 단계별 보여주기
[33~45초] 혜택 나열    — 핵심 3가지 (빠르게)
[45~52초] 사회적 증거  — 후기 발췌 or 수치
[52~60초] CTA + 긴급성 — 한정 수량/기간 + 링크
```

## 스타일별 구조 변형

### UGC (후기형)
```
[0~3초]  "솔직히 안 믿었는데요..." (진짜 사람 같은 어조)
[중반]   실제 사용 장면 (핸디캠/셀카 스타일)
[후반]   "이거 진짜 돼요" + 자연스러운 CTA
```

### 비포/애프터형
```
[0~3초]  Before 상태 (문제 상황)
[중반]   제품 사용 장면
[후반]   After 상태 (결과) → CTA
규칙: Before와 After는 같은 앵글, 같은 조명에서 찍어야 신뢰감
```

### 비교형
```
[0~3초]  "기존 ○○ vs 이 제품"
[중반]   나란히 비교 시연
[후반]   차이 강조 + CTA
규칙: 경쟁사 이름은 직접 언급 금지 → "일반 제품들" 표현
```

### 문제해결형 (전환율 최고)
```
[0~3초]  고통 상황 직접 묘사 ("이런 거 있죠?")
[중반]   "이거 하나면 해결됩니다" + 시연
[후반]   해결된 상태 + CTA
```

## 스크립트 작성 규칙

1. **구어체**: 블로그와 다르다. 말하듯이 쓴다.
2. **짧은 문장**: 한 문장 = 1초. 15자 이내.
3. **동사로 시작**: "써봤어요", "해봤더니", "보세요"
4. **숫자는 구체적으로**: "많이" → "2주 만에", "여러 번" → "하루 3번"
5. **자막 고려**: 핵심 키워드는 자막으로 강조될 것을 가정하고 쓴다
