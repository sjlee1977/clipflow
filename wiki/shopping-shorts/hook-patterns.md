# 쇼핑 쇼츠 훅 패턴 7종

첫 3초가 전부다. 아래 7가지 패턴 중 하나를 반드시 사용한다.

---

## 1. 금지/경고형 (Forbidden)
"이거 절대 사지 마세요..."
→ 역설적으로 관심을 끈다. 뒤에 "...이 더 좋은 게 있어서요"로 이어짐.

**적합한 제품**: 뷰티, 건강식품, 가전
**주의**: 과한 어그로는 역효과. 이어지는 내용이 납득 가능해야 함.

---

## 2. 수치 충격형 (Number Shock)
"2주 만에 -5kg", "3만 명이 재구매한 이유"
→ 구체적 숫자가 추상적 설명보다 10배 강하다.

**규칙**:
- 숫자는 최대한 구체적으로 (2주, 3만 명, 4.9점)
- 과장 금지 — 신뢰 가능한 수치만
- 숫자 뒤에 이유를 바로 연결

---

## 3. 공감 질문형 (Empathy Question)
"이런 거 있지 않아요? 샀는데 쓰지도 않고 자리만 차지하는 거..."
→ 시청자가 "맞아, 나도!"를 느끼는 순간 스크롤을 멈춘다.

**적합한 제품**: 생활용품, 다이어트, 정리용품
**구조**: 공감 질문 → "저도 그랬어요" → 해결책 등장

---

## 4. 비밀 폭로형 (Secret Reveal)
"피부과 원장이 직접 쓰는 세럼", "공장에서 직접 받아왔어요"
→ 내부자 정보, 유명인 사용, 숨겨진 사실 프레임

**적합한 제품**: 뷰티, 건강, 식품
**주의**: 사실이어야 한다. 검증 불가한 주장 금지.

---

## 5. 비포/애프터 예고형 (Before/After Tease)
첫 3초에 After 상태를 먼저 보여주고, "어떻게 됐는지 봐요"
→ 결과를 먼저 보여줌으로써 끝까지 보게 만든다.

**규칙**: After 장면은 극적일수록 좋다. 단 실제 결과여야 함.

---

## 6. 비교 도발형 (Comparison Challenge)
"기존 제품들이랑 뭐가 달라요? 직접 비교해봤어요"
→ 비교 영상은 시청 완료율이 높다.

**적합한 제품**: 가전, 청소용품, 주방용품
**주의**: 경쟁사 직접 언급 금지 → "일반 ○○들"로 표현

---

## 7. 긴급성 훅 (Urgency Hook)
"오늘만 이 가격이에요", "재고 30개 남았어요"
→ FOMO(놓치면 손해) 심리 활용

**규칙**:
- 반드시 사실이어야 한다 (거짓 희소성은 신뢰 파괴)
- 구체적 기한/수량 명시
- 상시 사용 금지 — 특별 행사 때만

---

## 훅 선택 가이드

| 제품 유형 | 추천 훅 패턴 |
|----------|------------|
| 뷰티/스킨케어 | 비밀 폭로형, 수치 충격형 |
| 다이어트/건강 | 수치 충격형, 공감 질문형 |
| 생활/청소용품 | 비포/애프터형, 비교 도발형 |
| 패션/의류 | 비포/애프터형, 공감 질문형 |
| 식품/음료 | 공감 질문형, 비밀 폭로형 |
| 가전/IT | 비교 도발형, 수치 충격형 |
| 한정 세일 | 긴급성 훅 |
