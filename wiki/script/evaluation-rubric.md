# 대본 채점 루브릭 (LLM Judge 기준)

이 루브릭은 YouTube 대본을 8개 차원에서 각 10점 만점으로 평가한다.
총점 = 8개 점수 합산 (80점 만점). 백분율로 환산 시 ×1.25.

---

## 1. 훅 강도 (Hook Power) — 10점

**평가 질문**: 첫 문장이 시청자를 멈추게 하는가?

| 점수 | 기준 |
|------|------|
| 9~10 | A/B/C형 훅 중 하나. 인사말 없음. 즉각적 긴장/궁금증 유발. |
| 6~8  | 훅 의도는 있지만 힘이 약하거나 너무 길다. |
| 3~5  | 배경 설명이나 개요로 시작. 시청자가 이탈할 동기 없음. |
| 0~2  | "안녕하세요", "오늘도 돌아왔습니다"로 시작. |

→ 훅 유형 상세: `hook-writing.md`

---

## 2. 7단계 구조 준수 (Structure) — 10점

**평가 질문**: 도입→팩트→개념번역→인사이트→시나리오→리스크→클로징의 7단계가 있는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 7단계 흐름이 명확. 단계 간 브릿지 멘트 존재. |
| 6~8  | 구조는 있지만 일부 단계가 빠지거나 순서가 어색하다. |
| 3~5  | 정보 나열형. 흐름이 없고 단계 간 연결이 없다. |
| 0~2  | 구조 없음. 단순 정보 덩어리. |

→ 구조 상세: `_shared/7-stage-structure.md`

---

## 3. 비유/개념 번역 (Analogy Translation) — 10점

**평가 질문**: 전문 용어를 일상 비유로 번역했는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 핵심 개념마다 일상 비유 존재. "초등학생도 고개를 끄덕이는" 수준. |
| 6~8  | 일부 비유 있지만 전문용어가 번역 없이 남아있다. |
| 3~5  | 비유 거의 없음. 전문 용어 나열. |
| 0~2  | 개념 번역 전무. 전부 전문 용어. |

→ 비유 기법 상세: `narrative-techniques.md`

---

## 4. 감정 흐름 (Emotional Arc) — 10점

**평가 질문**: 긴장→공감→놀람→각성→행동의 감정 흐름이 설계되어 있는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 5단계 흐름이 뚜렷. 중반부에 놀람 포인트 존재. |
| 6~8  | 일부 단계 존재. 흐름이 약하거나 단계 중 일부가 빠짐. |
| 3~5  | 감정 설계 없음. 정보만 전달. |
| 0~2  | 감정 흐름 전무. 건조하고 강의체. |

→ 감정 흐름 상세: `emotional-flow.md`

---

## 5. 리스크 구간 (Risk Balance) — 10점

**평가 질문**: 리스크를 정직하게 제시하고, 대응 안전망도 함께 제공했는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 리스크 2개 이상 + 구체 수치 + 대응 안전망 세트 완비. |
| 6~8  | 리스크 있지만 수치가 없거나 대응책이 약하다. |
| 3~5  | 리스크 1개이거나 단순 언급 수준. |
| 0~2  | 리스크 없음. 좋은 것만 나열. |

→ 리스크 구간 상세: `_shared/7-stage-structure.md` 6단계

## 6. CTA 품질 (Call To Action) — 10점

**평가 질문**: CTA가 자연스럽고 당당하게 핵심 인사이트 직후에 배치됐는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 시그니처 CTA 존재. 강요 없이 당당함. 제작 노력 어필 포함. |
| 6~8  | CTA 있지만 타이밍이 어색하거나 너무 애원하는 톤. |
| 3~5  | CTA가 맨 끝에만 형식적으로 존재. |
| 0~2  | CTA 없거나 "구독해주시면 감사하겠습니다" 수준. |

→ CTA 상세: `cta-writing.md`

---

## 7. 금지 표현 준수 (Forbidden Phrases) — 10점

**평가 질문**: 이탈을 부르는 금지 표현이 없는가?

금지 표현 목록:
- "안녕하세요" 도입부
- "오늘은 ~에 대해 알아보겠습니다"
- "오늘도 돌아왔습니다"
- "지금까지 ~알아보았습니다"
- "도움이 됐으면 좋겠습니다"
- "이상으로 마치겠습니다"
- "살펴보겠습니다", "확인해보겠습니다"
- "다양한 방면에서", "여러 가지 이유로"
- "우선 먼저" (중복어)

| 점수 | 기준 |
|------|------|
| 10   | 금지 표현 0개 |
| 7    | 금지 표현 1개 |
| 4    | 금지 표현 2개 |
| 0~2  | 금지 표현 3개 이상 |

→ 전체 금지 표현 + 페르소나: `writer-persona.md`

## 8. 인간 구술감 (Human Voice) — 10점

**평가 질문**: AI가 쓴 것처럼 느껴지지 않는가? 구어체와 리듬이 살아있는가?

| 점수 | 기준 |
|------|------|
| 9~10 | AI 패턴 0개 + 구어체 자연스러움 + 문장 길이 다양. 사람이 말하는 것 같다. |
| 7~8  | AI 패턴 1~2개이나 전반적으로 자연스러움. |
| 4~6  | AI 패턴 다수(3~5개) 또는 강의체 구간이 많다. |
| 1~3  | 전체가 "살펴보겠습니다" 구조, 나열식, 무감정. |
| 0    | 거의 모든 문장이 AI 보고서체. |

구현 체크리스트:
- [ ] 반말 대화체 일관 사용 (카테고리 설정에 따라)
- [ ] 짧은 문장 활용 ("그게 문제였어. 진짜로.")
- [ ] 질문으로 속도 조절 ("이게 무슨 의미인지 알아?")
- [ ] 브릿지 멘트 자연스러움 ("자, 여기서 끝내면 안 되지.")
- [ ] 문장 길이 불규칙 (1줄짜리와 4줄짜리 섞임)

## 총점 계산

총점 = 8개 차원 점수 합산 (80점 만점)

| 등급 | 점수 | 의미 |
|------|------|------|
| S   | 72~80 | 즉시 촬영 가능. 전문 작가 수준. |
| A   | 60~71 | 소폭 수정 후 촬영 가능. |
| B   | 48~59 | 핵심 약점 1~2개 수정 필요. |
| C   | 32~47 | 구조적 재작성 필요. |
| D   | 0~31  | 전면 재작성 권장. |

**리파이너 실행 조건**: `totalScore < 64` AND `약점 차원(score < 7) >= 1`
**최대 리파이너 라운드**: 2회
