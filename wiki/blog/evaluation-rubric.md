# 블로그 글 채점 루브릭 (LLM Judge 기준)

이 루브릭은 블로그 글을 8개 차원에서 각 10점 만점으로 평가한다.
총점 = 8개 점수 합산 → 100점 환산.

---

## 1. 훅 강도 (Hook Power) — 10점

**평가 질문**: 첫 문장이 독자를 멈추게 하는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 훅 A/B/C형 중 하나. 첫 문장이 20자 이내로 강렬하다. "안녕하세요" 없음. |
| 6~8  | 훅 의도는 있지만 힘이 약하거나 길다. |
| 3~5  | 배경 설명이나 개요로 시작. 독자가 두 번째 문장을 읽을 동기 부족. |
| 0~2  | "안녕하세요", "오늘은 ~에 대해 알아보겠습니다"로 시작. |

---

## 2. 서사 구조 (Narrative Structure) — 10점

**평가 질문**: 5막 구조(훅→갈등→반전→증명→해소)가 있는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 5막 흐름이 명확. 소제목만 읽어도 서사가 파악된다. |
| 6~8  | 구조는 있지만 일부 막이 빠지거나 순서가 어색하다. |
| 3~5  | 정보 나열형. 흐름이 없고 섹션 간 연결이 없다. |
| 0~2  | 구조 없음. 단순 정보 덩어리. |

---

## 3. 장면 묘사 / Show Don't Tell — 10점

**평가 질문**: "보여주기"를 사용하는가, 아니면 "말하기"만 하는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 구체적 장면, 시간, 행동 묘사. "그는 피곤했다" 같은 직접 서술 없음. |
| 6~8  | 일부 장면 있지만 추상적 서술과 섞여 있다. |
| 3~5  | 대부분 추상적. "많은 사람들이", "일반적으로" 같은 뭉뚱그린 표현 많음. |
| 0~2  | 장면 묘사 전무. 전부 선언적 서술. |

---

## 4. 문장 리듬 (Sentence Rhythm) — 10점

**평가 질문**: 짧은 문장과 긴 문장이 의도적으로 섞여 리듬이 살아있는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 핵심 문장은 10자 이내, 설명 문장은 적당히 길다. 리듬 변화가 느껴진다. |
| 6~8  | 어느 정도 변화 있지만 단조로운 구간이 있다. |
| 3~5  | 문장 길이가 균일하게 길거나 짧다. |
| 0~2  | 모든 문장이 비슷한 길이. 읽히는 속도감이 없다. |

---

## 5. 감정 흐름 (Emotional Arc) — 10점

**평가 질문**: 긴장→공감→놀람→안도→행동의 감정 흐름이 설계되어 있는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 5단계 흐름이 뚜렷. 3번째 단락 근처에 놀람 포인트 존재. |
| 6~8  | 일부 단계 존재. 흐름이 약하거나 단계 중 일부가 빠짐. |
| 3~5  | 감정 설계 없음. 정보만 전달. |
| 0~2  | 감정 흐름 전무. 건조하고 사무적. |

---

## 6. CTA 품질 (Call To Action) — 10점

**평가 질문**: CTA가 1개이고, 핵심 인사이트 직후의 적절한 타이밍에 배치되었는가?

| 점수 | 기준 |
|------|------|
| 9~10 | CTA 정확히 1개. "독자가 가장 고마워하는 순간" 직후 배치. 구체적 행동 요청. |
| 6~8  | CTA 있지만 타이밍이 어색하거나 너무 일반적("구독해주세요"). |
| 3~5  | CTA가 글 맨 끝에만 있거나 2개 이상. |
| 0~2  | CTA 없음 또는 "도움이 됐으면 좋겠습니다" 같은 무력한 마무리. |

---

## 7. 금지 표현 준수 (Forbidden Phrases) — 10점

**평가 질문**: 아마추어 글쓰기 신호인 금지 표현이 없는가?

금지 표현 목록:
- "안녕하세요" 도입부
- "오늘은 ~에 대해 알아보겠습니다"
- "이번 포스팅에서는"
- "지금까지 ~알아보았습니다"
- "도움이 됐으면 좋겠습니다"
- "이상으로 마치겠습니다"
- "다양한 방면에서", "여러 가지 이유로"
- "우선 먼저" (중복어)

| 점수 | 기준 |
|------|------|
| 10   | 금지 표현 0개 |
| 7    | 금지 표현 1개 |
| 4    | 금지 표현 2개 |
| 0~2  | 금지 표현 3개 이상 |

---

## 8. 클로징 에코 (Closing Echo) — 10점

**평가 질문**: 마지막 단락이 첫 단락의 장면이나 표현을 되울리며 완결감을 주는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 첫 단락의 장면/표현이 결말에서 해소된다. 독자가 완결감을 느낀다. |
| 6~8  | 주제는 연결되지만 구체적 에코가 약하다. |
| 3~5  | 결말이 첫 단락과 무관하다. 단순 요약으로 끝난다. |
| 0~2  | "지금까지 알아보았습니다" 같은 공식적 마무리. |

---

## 총점 계산

총점 = (8개 차원 합산 / 80) × 100

| 등급 | 점수 | 의미 |
|------|------|------|
| S   | 90~100 | 발행 즉시 가능. 전문 작가 수준. |
| A   | 75~89  | 소폭 수정 후 발행 가능. |
| B   | 60~74  | 핵심 약점 1~2개 수정 필요. |
| C   | 40~59  | 구조적 재작성 필요. |
| D   | 0~39   | 전면 재작성 권장. |
